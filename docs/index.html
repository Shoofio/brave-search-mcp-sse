Brave Search MCP SSE Helm Repository

This is a Helm chart repository for the Brave Search MCP SSE project. The chart includes:

  • Brave Search API integration
  • SSE transport for MCP
  • Prometheus metrics
  • Kubernetes-ready deployment

Usage

To use this Helm chart repository, add it to your Helm:

helm repo add brave-search-mcp-sse https://shoofio.github.io/brave-search-mcp-sse
helm repo update

Then you can install the chart with:

helm install my-release brave-search-mcp-sse/brave-search-mcp-sse
